# Security Policy

## Scope

ClawZero is designed for defensive security research and production execution-boundary enforcement.

Do not use this project to attack systems you do not own or explicitly control with authorization.

## Reporting a Vulnerability

Please report security issues privately to: security@mvar.dev

Include:

- vulnerability description
- affected version/commit
- reproduction steps
- impact assessment
- suggested remediation (if available)

## Disclosure Process

- We acknowledge reports as quickly as possible.
- We validate and triage severity.
- We coordinate a fix and release.
- We publish advisory notes after remediation.
