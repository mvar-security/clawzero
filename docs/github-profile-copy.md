## Suggested Pinned Discussion/Issue

### Title

What ClawZero Is / Is Not

### Body

ClawZero is an in-path enforcement substrate for production agent flows.

What it is:

- deterministic sink control before execution
- MVAR-backed enforcement decisions
- signed witness artifacts for auditability

What it is not:

- a red-team toolkit
- an attack simulation platform first
- an LLM-as-judge safety layer

The attack demo exists as proof of enforcement behavior.
The product is the boundary.
